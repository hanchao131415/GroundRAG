# 项目一环境配置（实测可用，版本锁定）

> 本文档记录 py312 环境的**实测可用**配置。所有版本号都是踩坑后验证过的,照着装能跑通。
> 详见《踩坑实录》坑14/15(Python 3.14 + transformers 5.x 的依赖地狱)。

---

## 一、为什么用这套配置

| 决策 | 原因 |
|---|---|
| **Python 3.12**(非3.14) | AI 生态对 3.12 支持最完善;3.14 太新导致 ragas/FlagEmbedding/tokenizers 编译失败 |
| **transformers 4.46.3**(非5.x) | 5.x 移除了 `prepare_for_model`,FlagEmbedding(reranker)报错 |
| **conda 环境**(非venv) | conda 管理 Python 版本更干净,可并存多版本 |
| **版本全部锁定** | 避免后续装包时依赖被连带升级,破坏兼容 |

---

## 二、环境准备(一次性)

### 1. 用 conda 创建 py312 环境
```bash
conda create -n py312 python=3.12 -y
conda activate py312
```

### 2. 确认版本
```bash
python --version
# 应输出: Python 3.12.x
```

## 三、安装依赖(按顺序,顺序重要)

> ⚠️ transformers 必须最后单独装并钉版本,否则会被别的包拽到 5.x。

### Step 1: LangChain 核心栈
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple \
  "langchain==0.3.26" \
  "langchain-core==0.3.86" \
  "langchain-community==0.3.27" \
  "langchain-text-splitters==0.3.8" \
  "langchain-huggingface==0.3.1" \
  "langchain-openai==0.2.14" \
  "langchain-anthropic==0.3.3"
```

### Step 2: 检索 + 文档处理
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple \
  "faiss-cpu" \
  "rank_bm25" \
  "PyMuPDF>=1.24" \
  "python-docx" \
  "jieba" \
  "python-dotenv" \
  "fastapi" \
  "uvicorn"
```

### Step 2.5: API 服务依赖
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple \
  "PyJWT>=2.8" \
  "slowapi>=0.1.9"
```
> PyJWT 用于 JWT 认证(`app/auth.py`),slowapi 用于 API 限流(`app/api.py` 的 `@limiter.limit`)。

### Step 3: FlagEmbedding(reranker)
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple FlagEmbedding
```

### Step 4: ⭐ 钉死 transformers 版本(关键!)
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple "transformers==4.46.3"
```
> 这一步**必须在 FlagEmbedding 之后单独做**。FlagEmbedding 装时会带最新 transformers(5.x),必须强制降回 4.46.3。

### Step 5: 压测工具(可选)
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple "locust>=2.20"
```
> locust 是压测工具,只在压测时需要。安装后用 `locust -f locustfile.py` 启动。

## 五、配置 .env

复制 `.env.example` 为 `.env`,填真实 key(见坑1:`=` 只能一个,base_url 带 `/`):

```bash
LLM_PROVIDER=zai
LLM_BASE_URL=https://api.z.ai/api/anthropic
LLM_API_KEY=你的z.ai真实key
LLM_MODEL=glm-4.6
LLM_TEMPERATURE=0.1
LLM_MAX_TOKENS=2048

# JWT 认证（生产必须改 JWT_SECRET！）
JWT_SECRET=rag-dev-secret-change-me-in-production
JWT_EXPIRE_HOURS=24

# LLM 降级链（可选，主LLM挂了自动切备用）
# LLM_FALLBACK_PROVIDERS=qwen,zhipu
# LLM_FALLBACK_API_KEYS=sk-key1,sk-key2
# LLM_FALLBACK_MODELS=qwen-plus,glm-4-flash
# LLM_FALLBACK_BASE_URLS=https://dashscope.aliyuncs.com/compatible-mode/v1,https://open.bigmodel.cn/api/paas/v4/
```

> ⚠️ key 只填 `.env`(不提交 git),绝不填 `.env.example`(会泄露)。
> ⚠️ 生产环境必须把 `JWT_SECRET` 改成随机字符串(至少32字符),可用 `python -c "import secrets; print(secrets.token_urlsafe(32))"` 生成。

## 七、常见问题

**Q: 装完 FlagEmbedding,reranker 报 `prepare_for_model` 错误?**
A: transformers 被拽到 5.x 了。`pip install transformers==4.46.3` 强制降级。

**Q: Python 3.14 装不了某些包?**
A: 换 3.12。3.14 太新,AI 生态未适配。

**Q: 中文 BM25 检索结果不对?**
A: 默认空格分词,中文失效。必须传 jieba 分词器(见检索模块)。

**Q: z.ai 报余额不足 429?**
A: 看 error code 1113(余额)而非 HTTP 429(限流)。充值或换模型。

**Q: 怎么测试 JWT 认证是否生效?**
A: 先 `POST /auth/login {"user_id":"zhangsan"}` 拿 token,然后请求带 `Authorization: Bearer <token>` 头。不带 token 或 token 过期会返回 401。
